Política de Privacidad

Skill de Alexa: Eventos Murcia

Términos de uso

Resumen

Esta Política de Privacidad explica cómo la skill de Alexa Eventos Murcia trata los datos del usuario. La skill ofrece información de eventos públicos a partir de fuentes abiertas y no solicita, recopila ni almacena datos personales del usuario fuera de los servicios de Amazon Alexa.

Última actualización: 25 de septiembre de 2025

Qué datos se tratan

  • Datos proporcionados a través de Alexa: las frases que digas a tu dispositivo se procesan por Amazon para reconocer la intención. La skill recibe solo la intención y los parámetros necesarios (por ejemplo, una fecha) para responder.
  • Datos personales propios: la skill no crea cuentas propias, no solicita nombre, email, ubicación precisa ni otra información personal.
  • Permisos opcionales: si habilitas funciones como recordatorios, la skill solicitará el permiso de Alexa correspondiente y usará exclusivamente la API de Amazon para programarlos. No se almacenan estos datos fuera de Alexa.

Finalidades

Prestar el servicio de consulta de eventos (por fecha, fin de semana, etc.) y, si lo autorizas, programar recordatorios a través de la plataforma de Alexa.

Fuentes de datos y terceros

  • Amazon Alexa: procesamiento de voz, intents y APIs (por ejemplo, Recordatorios). Sujeta a las políticas de Amazon.
  • Fuentes públicas de eventos: la skill consulta un feed público de eventos para generar respuestas.

Conservación

La skill no almacena datos personales persistentes. La información técnica en memoria se mantiene solo durante la sesión para poder responder (por ejemplo, la lista de eventos mostrada).

Seguridad

Se siguen prácticas estándar de desarrollo para mantener la seguridad del servicio. Las operaciones que impliquen datos (como recordatorios) se realizan a través de APIs oficiales de Amazon.

Derechos

Puedes gestionar tu consentimiento y permisos desde la app de Alexa (por ejemplo, revocar permisos de recordatorios). Para cualquier duda, contáctanos por email.

Cambios en esta política

Podemos actualizar esta Política para reflejar mejoras del servicio o cambios legales. Publicaremos siempre la versión vigente en esta URL.
